Sistema Operacional

Sistema Operacional é um programa ou um conjunto de programas cuja função é gerenciar os recursos do sistema (definir qual programa recebe atenção do processador, gerenciar memória, criar um sistema de arquivos, etc.), fornecendo uma interface entre o computador e o usuário ou utilizador.

Fundamentalmente, um sistema operacional é um software, que pode ser o Linux, Windows, Android, macOS, UNIX, entre outros. No entanto, ele não resume aquilo que seus olhos conseguem ver ou ao que você consegue interagir. Em outras palavras, é um programa que conversa diretamente com o hardware da sua máquina. Tipos de interface:

A GUI (Interface Gráfica do Usuário) é a interface mais comumente encontrada em sistemas operacionais modernos, como Windows, macOS e muitas distribuições de Linux. Ela utiliza elementos visuais, como janelas, ícones, menus e botões, para representar programas, arquivos e operações do sistema, simplificando a interação do usuário com o computador, permitindo que operações complexas sejam realizadas através de cliques do mouse e arrastar e soltar, em vez de comandos de texto.

A CLI (Interface de Linha de Comando) é uma interface baseada em texto na qual os usuários interagem com o sistema operacional por meio de comandos digitados em um prompt de comando.Exemplos de sistemas operacionais que suportam CLI incluem Windows (através do Command Prompt ou PowerShell) e todas as distribuições de Linux e UNIX (através do Terminal).Embora a CLI possa parecer mais complexa para usuários inexperientes, ela oferece mais controle e flexibilidade para usuários avançados e administradores de sistemas.

Algumas interfaces de sistema operacional são baseadas em menus de texto, nos quais os usuários navegam por diferentes opções e seleções usando teclas de seta e teclas de função. Essas interfaces são menos comuns hoje em dia, mas eram populares em sistemas operacionais mais antigos, como MS-DOS.
